# CRISPR-seq-analysis
## Seq Logo

#### 代码运行方式如下：

```shell
usage: analysis.py weblogo [-h] [-s SAMPLE] [-o OUT_PATH] [-p PROCESSES]

optional arguments:
  -h, --help            show this help message and exit
  -s SAMPLE, --sample SAMPLE
                        Sample information file.
  -o OUT_PATH, --out-path OUT_PATH
                        Result output path.
  -p PROCESSES, --processes PROCESSES
                        Multiple processes.
```
